## Quick Tips

### Technical Interview
- Explain observability pillars clearly
- Know the difference between monitoring and observability
- Understand ML model selection for different problems
- Be ready to whiteboard alerting system design

### Coding Assessment
- Time-series manipulation with pandas
- Basic anomaly detection algorithms
- Log parsing and analysis
- API integration patterns

### System Design
- Start with requirements gathering
- Consider scale and fault tolerance
- Discuss trade-offs explicitly
- Include monitoring for your design!
